# **App Name**: JL2G Costing Platform

## Core Features:

- Project Creation Wizard: Guide users through setting up the contractual, security, and financial framework of a new tender.
- Settings Management: Enable administrators to manage the master lists of personnel, materials, G&A cost tables, and milestone templates.
- Cost Build-Up: Allow tender leads to add, edit, and delete an unlimited number of labor line items.
- Material Costs: Enable tender leads to add unlimited material line items from a preset material database.
- Dashboards & Reporting: Provide access to real-time, auto-generated reports, including an Executive Summary, Cash Flow Summary, Profit & Loss statement, and a visual Timeline (Gantt chart).
- Scenario Planner: Allow users to model different cost scenarios by changing key variables and seeing the immediate impact on the project's Total Quote Value.
- AI Project Starter (Future): Use an AI tool to create a draft project based on a short user description.

## Style Guidelines:

- Primary color: Dark navy/charcoal (#131A35) for a professional, corporate feel.
- Accent color: Bright red/orange (#F96449) for call-to-action elements and highlights.
- Background color: Light grey (#F0F0F0) for a clean, neutral backdrop.
- Body and headline font: 'Inter' sans-serif for information clarity and data density. 
- Note: currently only Google Fonts are supported.
- Use minimalist, professional icons to represent different project elements and actions.
- Prioritize information clarity and data density over excessive whitespace.
- Subtle animations and transitions to enhance user experience without being distracting.
